# EKS Platform Project

This repository documents the step-by-step build of a production-ready Kubernetes platform on AWS using modern DevSecOps practices.

## Project Goals
- Provision AWS infrastructure with Terraform
- Create an EKS cluster using eksctl
- Deploy ingress controllers
- Implement monitoring with Prometheus and Grafana


## Structure
Each folder represents a project phase/day and contains:
- Infrastructure or configuration code
- A detailed README explaining decisions

## Tech Stack
- AWS (VPC, EKS, EC2, IAM, S3, DynamoDB)
- Terraform
- eksctl
- Kubernetes
- Prometheus & Grafana
- Bash
